What is Artefact - Tattoo Piercing Studio Template

“Artefact - Tattoo Piercing Studio Template" is perfect if you like a minimal, clean and modern design. This template is ideal for professional tattoo, tattoo artist, piercing shop, tattoo studio, tattoo portfolio, tattoo piercing, tattoo salon, body art, barber and those who need an easy, attractive and effective way to share their work with clients.

If you would like to create a page that has a strong first impression and intrigues everyone, Artefact template is the tool you need. Let it impress you with the live preview page. Artefact template is written in valid and clean HTML & CSS3 code. It’s Super easy to customize so it’ll suit your needs.

We used Bootstrap Framework, Owl Carousel, Animate, Themify Icons, Popup Video, Magnific Popup Gallery, Google fonts in this template. Links are available in Sources and Credits field.


Installation and customization about

Artefact template is a static HTML template. Not WordPress. If you want to publish / install as HTML, you can edit and publish with any editor (brackets, notepad++ etc). For this, you need to have basic knowledge of HTML and CSS. When you open HTML extensions in any editor, you can view and edit the description titles I mentioned. (Add your images and content articles in the relevant parts of the template.) Then connect to FTP and send your files to the server. Your site will work. There is no need for any installation. Or if you want to make a custom panel for this template, you can easily customize it.
